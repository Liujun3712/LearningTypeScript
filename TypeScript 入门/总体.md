#TypeScript 了解   
  
###Hello world   
   
构建第一个程序
```
function greeter(person) {
    return "Hello, " + person;
}

let user = "Jane User";

document.body.innerHTML = greeter(user);
```
下面可以对于person的变量类型进行注释，约定变量类型为`string`：   
```
function greeter(person: string) {
    return "Hello, " + person;
}

let user = [0, 1, 2];

document.body.innerHTML = greeter(user);
```
